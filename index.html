Official jQuery Plugin for Avro Phonetic

Various efforts have been made to create a javascript version of Avro Phonetic, but none of them is 100% compatible to Avro Keyboardâ€™s phonetic scheme. So we have created this official version and made it a jQuery Plugin so that people can easily integrate Bangla typing facility to their sites without depending on a native IME. It is also the first platform independent implementation of Avro Phonetic, so if Avro Keyboard is not available, you can use this handy tool to type Bangla.
